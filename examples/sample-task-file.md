# Sample Task File

This file demonstrates how tasks are linked to quests.

## Tasks

- [ ] First task to complete
- [ ] Second task to complete  
- [ ] Third task to complete
- [ ] Fourth task (visible by default)
- [ ] Fifth task (hidden until earlier tasks completed)
- [ ] Sixth task
- [ ] Final task - quest complete!

## Notes

- Check off tasks by changing `[ ]` to `[x]`
- The Quest Board will detect changes and award XP
- Only the next 4 incomplete tasks are visible by default
