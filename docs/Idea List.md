# Quest Board - Idea List

Ideas and features that didn't make the MVP cut. These can be evaluated for future releases.

---

## High Priority (Consider for v1.1)

### Quest Dependencies
- [ ] Block quests until prerequisite quests are completed
- [ ] Visual indicator showing "locked" status
- [ ] Dependency chain visualization
- **Use case:** "Can't apply to Company B until I finish tailoring resume for Company A"

### Quest Templates
- [ ] Save common quest patterns as templates
- [ ] Quick-create from template library
- [ ] Share templates with community
- **Use case:** "Junior Dev Application" template pre-fills common fields

### Recurring Quests
- [ ] Daily/Weekly/Monthly repeat patterns
- [ ] Auto-regenerate on completion
- [ ] Streak tracking for recurring quests
- **Use case:** "Take out trash every Tuesday" or "Apply to 5 jobs per week"

### Quest Notes Enhancement
- [ ] Rich text editor for notes
- [ ] Attach files/images to quests
- [ ] Link to other Obsidian notes
- **Use case:** Keep interview prep notes attached to quest

---

## Medium Priority (v1.2+)

### Analytics Dashboard
- [ ] Historical trends over time
- [ ] Heatmap of quest completion
- [ ] Category breakdown pie charts
- [ ] Export data to CSV
- **Use case:** "Show me my productivity patterns over the last 3 months"

### Boss Battles
- [ ] Major milestones with multi-stage progress
- [ ] Multiple sub-quests required to defeat
- [ ] Special rewards/animations on completion
- **Use case:** "Get Hired" boss battle requires: 10 apps, 3 interviews, 1 offer

### Equipment System
- [ ] Unlock tools/bonuses as you level up
- [ ] Equip items that modify XP gains or unlock features
- [ ] Visual equipment display on character sheet
- **Use case:** "Resume Review Tool" gives +10% XP to dev job applications

### Sound Effects (Optional)
- [ ] Quest completion sounds
- [ ] Level up fanfare
- [ ] Victory music on milestones
- [ ] Boss battle music during Pomodoro
- **Note:** Must be OFF by default, opt-in only

### Quest Timer Integration
- [ ] Built-in Pomodoro timer per quest
- [ ] Track time spent on each quest
- [ ] Generate time reports
- **Use case:** "How much time did I spend on applications this week?"

---

## Lower Priority (Nice to Have)

### Party System
- [ ] Invite friends/family to shared quest boards
- [ ] See each other's progress
- [ ] Team milestones
- [ ] Competitive leaderboards (optional)
- **Use case:** Job hunting with a friend, household chores with roommates

### Guild System
- [ ] Join or create guilds (themed groups)
- [ ] Shared goals and achievements
- [ ] Guild chat/discussion
- **Use case:** "ADHD Dev Job Hunters" guild with shared tips and encouragement

### AI Quest Suggestions
- [ ] Claude API integration
- [ ] Analyze current quests and suggest next steps
- [ ] Generate quest descriptions from job URLs
- [ ] Smart prioritization recommendations
- **Use case:** "Based on your skills and goals, you should apply to these 5 companies"

### Mobile Companion App
- [ ] iOS/Android app for quick quest capture
- [ ] Sync with Obsidian vault
- [ ] Push notifications for reminders
- **Use case:** Add quest idea while out and about

### Quest Marketplace
- [ ] Share quest packs with community
- [ ] Download pre-made workflows
- [ ] Rate and review quest packs
- **Use case:** Download "Software Engineering Job Hunt" quest pack

### Custom Quest Types
- [ ] Not just 4 columns, user-defined workflows
- [ ] Custom status labels
- [ ] Different board layouts
- **Use case:** Agile team could use Backlog → To Do → In Progress → Review → Done

### Integrations
- [ ] Todoist import
- [ ] Notion import
- [ ] Export to Trello
- [ ] Calendar app sync (Google Calendar, Outlook)
- [ ] Slack/Discord webhooks for team updates

### Gamification Enhancements
- [ ] Daily login bonuses
- [ ] Challenge mode (complete X quests in Y days)
- [ ] Seasonal events (New Year's Resolution sprint, etc.)
- [ ] Pet/companion system (virtual pet levels up with you)
- [ ] Achievement showcase (display unlocked achievements)

## Rejected Ideas (Keeping for Reference)

### Auto-Apply Feature
**Idea:** Automatically submit job applications via scraping
**Why rejected:** Ethical concerns, low success rate, not the plugin's purpose

### Cryptocurrency Rewards
**Idea:** Earn crypto tokens for completing quests
**Why rejected:** Overcomplicates, introduces financial/legal concerns

### Video Game Integration
**Idea:** Sync with actual video games (level up in Skyrim = level up in Quest Board)
**Why rejected:** Too niche, API limitations, maintenance burden
