# Quest Board - Character Sprite Prompts

Prompts for AI image generation (e.g., Whisk, Midjourney, DALL-E).
All sprites should be **pixel art style**, **south-facing portrait**, **no background**, suitable for RPG game UI.

---

## Style Reference (from Paladin Tier 4)

The Paladin sprites establish the visual language:
- Pixel art style with clean edges
- Character facing south (toward viewer)
- Heroic standing pose
- Visible equipment defining the tier
- No background (transparent or solid color for easy removal)

---

## Tier Progression Philosophy

| Tier | Level Range | Visual Theme |
|------|-------------|--------------|
| 1 | 1-5 | Novice - Basic cloth/leather, simple tools |
| 2 | 6-12 | Apprentice - Reinforced gear, class-specific elements emerging |
| 3 | 13-17 | Journeyman - Partial armor/robes, distinctive class silhouette |
| 4 | 18-24 | Master - Full class regalia, ornate details, flowing elements |
| 5 | 25+ | Legendary - Elite gear, glowing effects, peak visual expression |
