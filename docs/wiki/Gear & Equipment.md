## The Blacksmith (Smelting)

The Blacksmith lets you upgrade gear by combining items.

### How Smelting Works

1. Select **3 items of the same tier**
2. The Blacksmith produces **1 item of the next tier up**
3. Output slot depends on input:
   - **Same slot all 3** â†’ Guaranteed same slot output
   - **Mixed slots** â†’ Random slot from input types

### Smelting Bonuses

| Condition | Bonus |
|-----------|-------|
| All 3 same slot | Output guaranteed that slot |
| All 3 same tier | Upgrade to next tier |
| Mixed tiers | Output = highest input tier (no upgrade) |

### Legendary Refinement

Smelting 3 **Legendary** items produces another Legendary with:
- Potentially better stats (re-rolled)
- **Gold cost: 4,000g**

> [!IMPORTANT]
> Legendary smelting is the only smelting that costs gold!

## Set Bonuses

Gear dropped from quests in the **same folder** belongs to the same **set**. Equip multiple pieces for bonuses!

### How Sets Work

1. Each quest folder creates a unique set
2. Gear from that folder is tagged with the set ID
3. Equip 2, 4, or 6 pieces to activate bonuses

### Bonus Thresholds

| Pieces | Typical Bonus |
|--------|---------------|
| **2 pieces** | Minor stat bonus |
| **4 pieces** | XP or gold bonus |
| **6 pieces** | Major bonus or title |

### AI-Generated Set Bonuses

If you have Gemini AI configured, set bonuses are **thematically generated** based on the folder name. For example:

- "Fitness" folder â†’ Strength/Constitution bonuses
- "Work Projects" folder â†’ XP bonus for Dev category

Without AI, default bonuses are applied.

### Viewing Active Set Bonuses

Check your Character Sheet to see:
- Currently active set bonuses
- How many pieces equipped per set
- Next bonus threshold
